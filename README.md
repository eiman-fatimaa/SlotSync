# SlotSync
# Academic Appointment and Scheduling System
A project by Maryam Amir, Eiman Fatima, Romaisa Kashif & Arham Nabi

In many universities, student-professor appointments are managed through informal communication channels such as emails, messaging applications or verbal communication. This often leads to double booking &amp; inefficient scheduling. 
We created an intelligent system that ensures a fair, conflict-free academic appointment management system.
